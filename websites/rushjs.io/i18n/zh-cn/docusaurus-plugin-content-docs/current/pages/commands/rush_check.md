---
title: rush check
---

```
用法： rush check [-h] [--variant VARIANT] [--json]

检查每个项目的 package.json 文件来确保仓库内所有的依赖都有相同的版本。

可选参数：
  -h, --help         展示帮助信息并退出
  --variant VARIANT  Rush 命令通过使用安装配置文件变量该参数可以通过 RUSH_VARIANT 环
                     境变量来指定。
  --json             该参数指定后会被输出 JSON 格式的文件
```
